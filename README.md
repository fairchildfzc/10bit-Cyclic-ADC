## Requirement Specifications

| Parameter  | Specification |
| :------------: |:---------------:| 
| Resolution      | 10-bit |
| Accuracy     | >9-bit       |  
| SFDR | >65dBc@2MHz       |  
| Sampling Rate | 5MHz       |  
| Analog Input Bandwidth | >5MHz       |  
| Input Range | 2V differential        |  
| Power Dissipation | <10mW      |  
| Operating Temperature | 0 - 80     |  
| Die Size | <2$mm^2$      |  
| Process | 0.18um CMOS       |
