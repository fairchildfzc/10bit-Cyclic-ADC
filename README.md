## Requirement Specifications

| Parameter  | Specification |
| :------------: |:---------------:| 
| Resolution      | 10-bit |
| Accuracy     | >9-bit       |  
| SFDR | >65dBc@2MHz       |  
| Sampling Rate | 5MHz       |  
| Analog Input Bandwidth | >5MHz       |  
| Input Range | 2V differential        |  
| Power Dissipation | <10mW      |  
| Operating Temperature | 0 - 80<sup>o</sup>C   |  
| Die Size | <2mm<sup>2</sup>      |  
| Process | 0.18um CMOS       |
